# ProStep Multi-Step Form

A professional, feature-rich multi-step registration form built with React, TypeScript, and Tailwind CSS. 

## Features

- ✅ **Multi-Step Flow**: Logical separation of personal info, account setup, and preferences.
- ✅ **Dynamic Progress Indicator**: Real-time visual feedback on completion status.
- ✅ **Smart Validation**: Instant validation for required fields and format checks.
- ✅ **Local Persistence**: Automatically saves progress to `localStorage` to prevent data loss.
- ✅ **Smooth Transitions**: CSS-powered animations for a premium user experience.
- ✅ **Final Summary**: Comprehensive review step before data submission.

## Tech Stack

- **Framework**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Run Locally

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.
